# Dev Club Hub

A community platform for student projects, mentors, alumni, and teams.

## What this repo contains

- Frontend built with React, Vite, TypeScript, and Tailwind
- Backend API with Express, Prisma, PostgreSQL, JWT auth, and validation
- Authentication, projects, teams, mentors, and user profile features
- Playwright test configuration and frontend build scripts

## Getting started

1. Install dependencies: `npm install`
2. Start the app: `npm run dev`
3. Build for production: `npm run build`
