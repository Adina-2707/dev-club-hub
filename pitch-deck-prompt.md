# Промпт для генерации Pitch Deck для Dev Club Hub

Создай pitch deck для платформы Dev Club Hub - образовательной платформы, соединяющей студентов, менторов и выпускников (алумни) для развития навыков в IT.

## Структура выступления (7 минут):

### 1. Титульный слайд (Title Slide)
- Название: Dev Club Hub
- Теглайн: "Соединяем поколения IT-специалистов для совместного роста"
- Логотип или иконка платформы
- Имя презентатора
- Дата

### 2. Проблема — что решали и для кого (~1 минута)
- Студенты не имеют доступа к реальным проектам и менторству
- Выпускники теряют связь с alma mater и не могут передавать опыт
- Отсутствие единой платформы для непрерывного обучения
- Целевая аудитория: студенты IT, менторы, алумни

### 3. Подход — как решили, на каком стеке (~1 минута)
- Frontend: React + TypeScript + Vite + Tailwind CSS
- Backend: Node.js + Express + Prisma + PostgreSQL
- Архитектура: REST API, аутентификация JWT, роли пользователей
- Ключевые фичи: профили, проекты, менторство, блог, вакансии

### 4. Трудности — про реальную проблему и как её решили (~1,5 минуты)
- Реальные проблемы: интеграция ролей, управление доступом, блокировка пользователей
- Решения: middleware для аутентификации, роли в базе данных, модальные окна для админ-функций
- Технические вызовы: работа с Prisma, валидация данных, UI/UX для разных ролей

### 5. Что дальше — что можно было бы доработать (~30 секунд)
- Дополнительные фичи: realtime чат, видео-звонки, аналитика
- Масштабирование: мобильное приложение, интеграции с внешними сервисами
- Улучшения: performance, security, user feedback

### 6. Живое демо (самая важная часть)
- Покажите работающий продукт: регистрация, профили, проекты, менторство
- Не тратьте много времени на объяснения — пусть аудитория увидит функционал

### 7. Спасибо за внимание (Thank You)
- "Спасибо за внимание!"
- Контакты для связи
- QR-код для доступа к MVP
- Призыв к действию: "Попробуйте Dev Club Hub"

## Дизайн:
- Современный, минималистичный стиль
- Цветовая схема: синий и зеленый (технологии + рост)
- Использовать иконки и графику для визуализации
- Каждый слайд должен иметь четкую структуру: заголовок, ключевые пункты, визуалы
- Ориентируйтесь на время выступления — слайды должны поддерживать рассказ, а не заменять его

## Формат:
Создай слайды в PowerPoint/Google Slides или Canva, используя этот промпт как основу. Фокус на живом демо как ключевой части презентации.
